# Вероятностные KAN для потоков тепла между океаном и атмосферой

Репозиторий с кодом для дипломной темы:

**«Вероятностные модификации нейросетей Колмогорова–Арнольда и их применение для анализа потоков тепла между океаном и атмосферой»**

Проект представляет собой переработку исходного ноутбука с baseline-моделью для SST в полноценный GitHub-репозиторий под задачу **прогноза пространственных полей потоков тепла** по данным ERA5.

---

## Идея работы

В репозитории реализована единая пространственно-временная архитектура, в которой:

- **encoder** общий для всех экспериментов;
- **temporal pooling** общий для всех экспериментов;
- **decoder body** общий для всех экспериментов;
- меняется только **выходная head**.

Это позволяет честно сравнивать вклад:

- обычной детерминированной головы;
- вероятностной головы;
- KAN/RBF-KAN головы;
- вероятностной KAN/RBF-KAN головы.

Главный принцип экспериментов:

```text
Input sequence -> CNN Encoder -> Temporal Attention Pooling -> Shared Decoder Body -> Output Head
```

То есть в этом проекте **мы не меняем decoder целиком**, а сравниваем **разные output head при неизменном decoder body**.

---

## Что предсказывает модель

Модель предсказывает **пространственные карты потоков тепла** на заданных горизонтах прогноза.

Поддерживаемые target-переменные:

- `LHFLX` — latent heat flux
- `SHFLX` — sensible heat flux
- `QNET` — net surface heat flux

Режимы target:

- `absolute` — предсказывать абсолютное значение потока
- `delta` — предсказывать изменение потока относительно последнего входного шага

Для дипломной работы рекомендуется использовать **`absolute`** как основной режим.

---

## Архитектура

### Общий backbone

Во всех экспериментах используются одинаковые блоки:

- `SmallEncoder` — сверточный spatial encoder;
- `TemporalAttentionPool` — агрегирование по времени;
- `SharedDecoderBody` — общая часть decoder.

### Сравниваемые output head

В репозитории уже включены следующие варианты:

- `conv_det` — детерминированная свёрточная голова
- `mlp_det` — детерминированная MLP-голова
- `rbf_kan_det` — детерминированная RBF-KAN голова
- `conv_gauss` — вероятностная Gaussian conv head
- `rbf_kan_gauss` — вероятностная Gaussian RBF-KAN head

Для вероятностных моделей предсказываются:

- `mu` — математическое ожидание поля потока;
- `sigma` — оценка неопределённости.

---

## Почему такая постановка удобна для диплома

Эта организация экспериментов хорошо соответствует теме диплома, потому что позволяет отдельно оценить:

1. даёт ли выигрыш **вероятностная постановка**;
2. даёт ли выигрыш **KAN/RBF-KAN head**;
3. даёт ли дополнительный эффект именно **вероятностная KAN-модификация**.

То есть сравнение получается методологически чистым:

- один и тот же backbone;
- один и тот же decoder body;
- меняется только head.

## Быстрый старт

### 1. Настройте конфиг

Откройте `configs/base.yaml` и укажите:

- путь к данным `data_dir`;
- карту файлов `file_map`;
- список входных каналов `channels`;
- target-переменную `target_key`;
- режим target `target_mode`.

### 2. Запустите один эксперимент

Пример для обычной детерминированной головы:

```bash
python scripts/train_experiment.py --config configs/base.yaml --head conv_det
```

Пример для вероятностной RBF-KAN головы:

```bash
python scripts/train_experiment.py --config configs/base.yaml --head rbf_kan_gauss
```

### 3. Запустите весь набор head-экспериментов

```bash
python scripts/run_head_sweep.py --config configs/head_sweep.yaml
```

## Набор экспериментов

Основная серия экспериментов:

| Эксперимент | Backbone | Decoder body | Head | Тип |
|---|---|---|---|---|
| `conv_det` | общий | общий | Conv | детерминированный |
| `mlp_det` | общий | общий | MLP | детерминированный |
| `rbf_kan_det` | общий | общий | RBF-KAN | детерминированный |
| `conv_gauss` | общий | общий | Conv | вероятностный |
| `rbf_kan_gauss` | общий | общий | RBF-KAN | вероятностный |

Таким образом:

- `conv_det` — базовый baseline;
- `rbf_kan_det` — показывает вклад KAN без вероятностной постановки;
- `conv_gauss` — показывает вклад вероятностной постановки без KAN;
- `rbf_kan_gauss` — основная модель диплома.

## Лоссы и метрики

### Для детерминированных моделей

Используются masked-метрики по океанической маске:

- `MAE`
- `RMSE`

### Для вероятностных моделей

Используется:

- masked Gaussian negative log-likelihood

Дополнительно можно считать:

- `MAE` по `mu`
- `RMSE` по `mu`
- калибровку доверительных интервалов
- покрытие доверительных интервалов

## Конфигурация

Главные настройки находятся в `configs/base.yaml`.

Наиболее важные поля:

```yaml
model:
  head_name: rbf_kan_gauss

data:
  target_key: LHFLX
  target_mode: absolute
  channels: [SST, U, V, LAT, LON]
```

Если хотите менять целевой поток, достаточно поменять:

```yaml
target_key: SHFLX
```

или

```yaml
target_key: QNET
```

## Возможные направления расширения

Этот репозиторий можно естественно развивать дальше:

- добавить `Student-t` вероятностную голову;
- добавить более общий `KANHead` вместо `RBF-KANHead`;
- добавить calibration metrics;
- добавить physics-aware regularization;
- добавить multi-target prediction (`LHFLX + SHFLX`);
- добавить прогноз нескольких горизонтов одновременно;
- заменить temporal block на T-KAN / state-space модуль.

## Рекомендуемая основная модель для диплома

Если нужен один главный кандидат для дипломной работы, рекомендуется использовать:

**`rbf_kan_gauss`**

То есть:

- общий CNN encoder;
- temporal attention pooling;
- общий decoder body;
- вероятностная RBF-KAN output head.

Именно эта модель лучше всего соответствует теме:

> вероятностные модификации нейросетей Колмогорова–Арнольда для анализа потоков тепла между океаном и атмосферой.

---

## Как оформить это в тексте диплома

Эту реализацию можно описывать так:

> В работе используется единая пространственно-временная архитектура, включающая сверточный пространственный encoder, модуль временной агрегации на основе attention и общую декодирующую часть. Для исследования вклада вероятностных модификаций нейросетей Колмогорова–Арнольда в финальном выходном блоке сравниваются детерминированные, вероятностные и KAN/RBF-KAN головы при фиксированном backbone.
